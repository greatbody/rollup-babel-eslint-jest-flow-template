# Introduction
This is a template to build nodejs app

# Pre-requirements

- VSCode installed
- VSCode Plugin Installed
  - ESLint
  - Flow Language Support
- Node Installed
- NPM Installed
- Git Installed

# Usage

> Step 1

```bash
git clone git@github.com:greatbody/rollup-babel-eslint-jest-flow-template.git
```

> Step 2

```bash
npm install
```
> Step 3

```bash
npm run build
```

> Step 4

```bash
node dist/index.js
```
